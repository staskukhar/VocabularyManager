# Authentication

## Description

- User must be logged in to use the app.

- When user is not logged in app should display login page not info is visibple.

- User must be able to log out.

- Use keycloak image for authentication. Integrate it with compose.

- Place all sensetive values outside in .env, describe them in env-example.

- Implement the simplest flow possible fro keycloak but it must follow good practices.

- Pay attention to defined rules.

- The admin user is being created manually and then user can login with provided credentials.
