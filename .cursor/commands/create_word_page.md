# Word page

## Description
- Page can be navigated by clicking the word from word list or search result, this page should load word data and its meanings.

- It must allow to edit and remove the word and its meanings. When word is being removed, it must return to previous URL.

- The edit logic should be moved from modal to the page. Also, the edit button of Word Plate should redirect this to the Word Page.

- Use FluentUI.
